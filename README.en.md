# 我的世界存档编辑工具

#### Description
本项目希望创建一组基于最新的.Net平台的易于使用、高效、精准、跨操作系统的Minecraft存档编辑工具。
计划优先对最新的Java版存档提供支持，未来可能会考虑支持基岩版或老版本，但不确定。
本项目不是 Minecraft 官方产品、不是来自 Minecraft 且未经 Minecraft 认可。
